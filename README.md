# 🌌 AetherOS

AetherOS é um Sistema Operacional experimental desenvolvido em **Java**, focado em modularidade, código aberto e na implementação de um sistema de arquivos de próxima geração, o **EXT5**.

## 🚀 Sobre o Projeto

O AetherOS busca explorar os limites da JVM (Java Virtual Machine) para gerenciamento de baixo nível. O objetivo principal é criar um ambiente estável e seguro, utilizando conceitos modernos de arquitetura de software e licenciamento permissivo.

### ✨ Diferenciais
* **Kernel Modular:** Gerenciamento de processos e drivers de forma isolada.
* **EXT5 File System:** Implementação experimental de um sistema de arquivos com Journaling nativo e Inodes dinâmicos.
* **Interface Shell:** Linha de comando intuitiva para interação direta com o Kernel.
* **Open Source:** Desenvolvido sob a licença MIT.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java 21+
* **Sistema de Arquivos:** EXT5 (Experimental)
* **Licença:** MIT

## 📁 Estrutura do Repositório

* `/src/kernel`: Código fonte do núcleo do sistema.
* `/src/fs`: Implementação do sistema de arquivos EXT5.
* `/src/shell`: Interface de comando do usuário.
* `/docs`: Documentação técnica e especificações do EXT5.
